# Advanced Capabilities (Secondary Surface)

This document tracks advanced analytics and AI capabilities that support the mission-control core but are not the primary operator entry point.

## Phase 2 QA Intelligence

- Test impact analysis
- Automated failure triage
- Suite hygiene scanning
- Deployment risk scoring
- Cross-RCA pattern detection
- Alert-driven test generation
- Test effectiveness scoring

## Platform Analytics

- Pareto analysis across churn, incidents, and quality dimensions
- Retention campaign generation
- Trend and effectiveness reporting

## Product Positioning Note

These capabilities remain important, but they are intentionally presented as secondary tabs/workflows to keep core operations focused on incident detection, actioning, and verified recovery.
